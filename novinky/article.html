Microsoft Visual C++ Distribuovatelné nejnovější podporované soubory. Visual C++ Redistributable nainstaluje knihovny runtime Microsoft C a C++ (MSVC). Tyto knihovny vyžaduje mnoho aplikací vytvořených pomocí nástrojů Microsoft C a C++. Pokud vaše aplikace tyto knihovny používá, musí být před instalací aplikace v cílovém systému nainstalovaný balíček Microsoft Visual C++ Redistributable. Architektura distribuovatelných balíčků se musí shodovat s cílovou architekturou vaší aplikace. Distribuovatelná verze musí být minimálně stejně aktuální jako sada nástrojů sestavení MSVC použitá k sestavení aplikace. Doporučujeme používat nejnovější redistribuci dostupnou pro vaši verzi sady Visual Studio s některými výjimkami uvedenými dále v tomto článku.
